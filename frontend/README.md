# Cozy Retro & Cafe Parlor

A beautifully crafted, high-performance culinary web interface celebrating organic wellness, customized nourishment, and cottagecore aesthetics. This client-side application brings together a warm, welcoming visual palette and sophisticated interactive dining controls, perfect for modern bistros and teahouses.

---

## ✨ Features

- **🎬 Full-Bleed Cinematographic Background**: A high-definition, uninhibited background video loop establishes an elegant, bright, and immersive parlor atmosphere on arrival.
- **🧁 Custom Menu Customizer**: A minimal, highly refined management interface allowing real-time curation of culinary profiles.
  - **Fluid Dynamic Sorting & Search**: Instant filter keys for calorie density, pricing structure, and categories.
  - **Sticky Sidebar Navigation**: A desktop layout that stays cleanly anchored to prevent blank space during scrolls.
  - **Bulk Template Importer**: Instantiate traditional Indian menus, french patisserie selections, or healthy brunch cards on demand.
  - **Data Resilience & Portability**: Inspect and copy full valid menu structures via structured JSON exports.
- **🍱 Dynamic Combo Creator**: An interactive tactile sandbox helping users assemble their custom wellness platters, pairing matching proteins, pastries, and artisanal matchas.
- **📅 Elegant Reservation Scheduler**: Fully featured table selection module mapped against seat counts and live-synchronized parlor occupancy metrics.
- **🛒 Micro-Cart & Active Food Tracker**: Drag, drop, add, and monitor live deliveries with simulated courier steps updated instantly on the map board.

---

## 🎨 Design Philosophy \& Aesthetic

This application deviates from cookie-cutter dashboards to establish an authentic, warm, and highly deliberate visual experience:
- **Color Palette**: Sophisticated ivory white cream base canvas (`#FFFDF9`) paired with subtle rosemary blossoms, elegant wild strawberry pinks, and deep tactile charcoal headings.
- **Typography**: Editorial-grade Serif typeface display headings paired with high-legibility sans-serif copy and JetBrains Mono markers.
- **Aesthetic Accents**: Subtle gingham overlays, elegant border treatments, and negative space to allow premium imagery and high-definition video loops to remain clear and sharp.

---

## 🛠️ Technological Stack

- **Framework**: [React 18](https://react.dev/) / [Vite](https://vite.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/) (Strictly typed schemas \& state interfaces defined in `src/types.ts`)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: Fluid layout transitions and reactive gestures powered by [motion/react](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 💻 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your workspace environment.

### Installation

1. Install applet dependencies:
   ```bash
   npm install
   ```

2. Direct local development execution:
   ```bash
   npm run dev
   ```
   *The local server typically boots directly on port `3000`.*

3. Production Compile:
   ```bash
   npm run build
   ```

4. Code Diagnostics:
   ```bash
   npm run lint
   ```
