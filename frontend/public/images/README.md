# Diner Landing Page Images folder

Place your images here in the `/public/images/` directory using the file explorer.
They will be available directly at the path: `/images/your-image-filename.png` in the application code.

### Loaded Assets:
We have prepared several beautiful, AI-generated pastel style images which you can find in `/src/assets/images/` as starting examples!
You can drop your own custom files here anytime (e.g., `burger.png`, `shake.png`, `donut.png`) and update the component properties.
